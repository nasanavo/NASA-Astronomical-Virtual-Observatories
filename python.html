Python Notebooks

We have created a number of Python notebooks showing concrete examples of how to query archives using VO protocols. We use utility functions we have written to make these interfaces simple and familiar to users of astroquery. We hope to integrate them into astroquery soon, but for now they are bundled with our demonstration notebook. The notebooks are summarized below. In every case, the idea is to allow you to query any relevant archive with the same interface, rather than having to become familiar with each archive's custom APIs.

These are available on our GitHub site at https://github.com/NASA-NAVO/notebooks. Clone the whole repository if you wish to run the notebooks yourself, or simply click on the links below to visualize them only.

Basic NASA-NAVO Services

  • Images: The Simple Image Access protocol allows you to download and visualize regions of the sky from a variety of sources. We demonstrate how to do this with our Image Access notebook.

  • Catalogs: The Simple Cone Search protocol performs simple searches of catalogs based on a given coordinate and radius. For more complicated queries, e.g., if you wish to search, match, and cross-correlate catalog data, the VO provides a more powerful Table Access Protocol. We demonstrate both in our Catalog Queries notebook that uses our Cone and TAP query classes.

  • Spectra: The Simple Spectral Access protocol allows you to download and visualize spectra of a given object from a variety of archives. We demonstrate how to do this within python in our Spectral Access notebook.

We have also run several workshops at AAS meetings on how to access NAVO services through python, and the notebooks we developed for the first workshop are still available. These predate our development of the utility classes, which have simplified the process considerably. But they are still available at the old notebook site. They include examples specific to each archive, while the above notebooks are generic.
